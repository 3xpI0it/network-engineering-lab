# Troubleshooting

## Device not communicating

Check:
- IP address
- Subnet mask
- Default gateway
- Cable connection
- Device status

---

## Ping not working

Check:
- Same network or different network
- Correct gateway
- Router configuration
- Interfaces are up (no shutdown)

---

## Basic troubleshooting approach

1. Check physical layer (cables, connections)
2. Check IP configuration
3. Check network reachability using ping
4. Verify router/switch configuration
