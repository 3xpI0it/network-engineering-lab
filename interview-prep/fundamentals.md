# Networking Fundamentals

## What is a Network?

A network is a group of connected devices that communicate and share data.

---

## What is a Switch?

A switch is a Layer 2 device that connects devices within the same network and forwards data using MAC addresses.

---

## What is a Router?

A router is a Layer 3 device that connects different networks and forwards data based on IP addresses.

---

## What is an IP Address?

An IP address is a unique identifier assigned to a device in a network.

---

## What is a Subnet Mask?

A subnet mask is used to determine whether a destination IP is in the same network or a different network.
