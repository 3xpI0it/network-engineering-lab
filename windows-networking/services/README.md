# Windows Network Services

This section covers essential network services in Windows environments.

It focuses on understanding how core services operate and their role in network communication and management.

---

## Purpose

- Learn the role of network services in Windows  
- Understand how services support network functionality  
- Build foundational knowledge for server-based networking  

---

## Topics Covered

- DHCP (Dynamic Host Configuration Protocol)  
- DNS (Domain Name System)  
- Introduction to directory services (basic level)  

---

## Key Concepts

- Automatic IP address assignment  
- Name resolution  
- Centralized network management  

---

## Approach

This section focuses on conceptual understanding first, followed by practical exploration of how these services are used in real environments.

---

## Goal

To build a foundation for working with Windows-based network services and prepare for advanced topics such as server configuration and enterprise networking.
